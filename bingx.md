# BingX Inverse (Coin-M) API Docs

## REST Trades Recovery
Tested on 2026-02-26. The URLs /openApi/cswap/v1/market/trades and /openApi/cswap/v1/quote/trades return "100400 this api is not exist". BingX standard swap (/openApi/swap/v2/quote/trades) only supports USDT-M and USDC-M. There is NO PUBLIC REST TRADES endpoint for BingX Inverse.
