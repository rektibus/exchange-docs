# BigONE OpenAPI Specifications

This repository contains the OpenAPI specifications (Swagger) for the BigONE Exchange API.

## Contents

- `spot_rest.yml`: Spot Trading API
- `contract_rest.yml`: Futures/Contract Trading API
- `wallet_rest.yml`: Wallet and Asset Management API
- `convert_rest.yml`: Convert/One-Click Swap API
- `spot_server.yml`: Server information for Spot API
